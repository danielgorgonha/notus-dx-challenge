# Notus DX Challenge - MVP & Roadmap Completo

## MVP Definição

**Produto**: Notus Web3 Wallet Dashboard - Uma interface unificada que demonstra todas as funcionalidades da API Notus através de um fluxo de usuário real.

**Core Features**:
- Autenticação Web3 via Privy (login social/email/wallet)
- Smart Wallet creation/management
- Dashboard com saldo e histórico
- Transfers entre carteiras
- Swaps de tokens
- Interação com Liquidity Pools
- KYC flow (se tempo permitir)

**Stack Técnico**:
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Privy SDK (@privy-io/react-auth)
- Notus API (REST)

## Contingency Plan

**Se alguma API não funcionar**:
- Documente o problema detalhadamente
- Implemente mock/placeholder
- Foque nas APIs que funcionam
- Use isso como feedback valioso

**Se ficar sem tempo**:
- Priorize: Auth → Transfers → Swaps
- Liquidity pools pode ser demonstrado via API calls no video
- KYC pode ser documentado como "future work"

**Se encontrar bugs críticos**:
- Documente com screenshots/logs
- Tente workarounds
- Continue com outras funcionalidades
- Relate no feedback como high priority
